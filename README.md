# Alif Fajdhan Yudhistiro - Personal Portfolio

Welcome to my personal portfolio website! This project is a modern, responsive, and beautifully designed web application built to showcase my skills, projects, and professional background.

## 🚀 Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** Vercel

## ✨ Features

- **Modern UI/UX:** Clean, dark-mode prioritized design with elegant gold/maroon accents.
- **Smooth Animations:** Page transitions, preloader, and hover effects using Framer Motion.
- **Dynamic Projects Showcase:** A sleek gallery of my latest development projects.
- **Interactive Certificates:** Direct links to my professional credentials.
- **Working Contact Form:** Email integration via FormSubmit with auto-reset functionality.
- **Fully Responsive:** Perfectly optimized for mobile, tablet, and desktop views.

## 💻 Running Locally

To run this project on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/alfdhzz/my-portofolio-website.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
